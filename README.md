# 📊 Power BI Data Professionals Dashboard

## 🧾 Project Overview
This project showcases an interactive Power BI dashboard built to analyze survey data from data professionals. The dashboard provides insights into salaries, job roles, skills, and career challenges within the data industry.

---

## 🛠️ Tools Used
- Power BI  
- Microsoft Excel  

---

## 📊 Key Features
- Salary analysis by job title  
- Country distribution of survey participants  
- Favourite programming languages among professionals  
- Salary comparison by gender  
- Difficulty level of breaking into the data field  
- Overall satisfaction with salary
